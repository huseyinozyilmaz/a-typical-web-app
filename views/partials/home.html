A Typical Web App

This web application presents various cases of DOM generation and manipulation to provide a test platform for end to end testing for frameworks like Selenium and Cucumber.

Each page listed below offers different behaviour and mimics typical web application page designs including synchronous form posts, asynchronous Ajax calls and dynamic Dom generation, etc...

Typical Web Pages

Login

Simple form that uses synchronous post to server and navigate to result page

Subscribe

Asynchronous form uses Ajax post to server and display the result by showing an existing dom element

Users

An Ajax record creation with some random delay from server. Server response generates new DOM elements

Greetings

An Ajax delayed list that gets generated with data returned from server. Content that does not fit in visible view.

Navigation

Simple navigation options to demonstrate different use cases with navigation

Alert

Uses default browser alert window to show a notification to users

Internet Usage

Tabular data presented in html table with client side filtering option

Large Cities

Uses default browser alert window to show a notification to users
